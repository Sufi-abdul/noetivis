
# Noetivis Demo Script (10–12 minutes)

1) Problem (1 min)
- Creators and businesses earn across platforms but reporting is fragmented.
- Partners/resellers lack one system to monetize services + track payouts.

2) Solution (1 min)
- Noetivis = intelligence + value ledger + automated payouts.

3) Live walkthrough (6–7 min)
- Show /health
- Login (DB auth): /auth/login-db
- Marketplace purchase: /purchase/marketplace-attr
- Founder lists payouts: /payouts/list?status=pending
- User earnings: /earnings/me
- Treasury + contributor pool distribution (Phase 10 endpoints)
- Show landing page and partner onboarding packet

4) Business model (1 min)
- API tiers + take-rate + white-label + revenue share.

5) Close (1–2 min)
- Early partners.
- Integrations roadmap.
