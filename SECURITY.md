
# Security Policy

## Supported Versions
Only the latest major release is supported with security updates.

## Reporting a Vulnerability
Please report security issues privately:
- Do NOT open a public GitHub issue for active vulnerabilities.
- Use your private channel / email to the core maintainers.

## Scope
- Authentication/authorization flaws
- Data leakage
- Supply chain issues
- RCE, injection, and privilege escalation
